# Osher Calisthenics

גרסה ראשונה יציבה לפרסום ב-GitHub Pages.

## איך מעלים
1. פתח את ה-ZIP.
2. היכנס לתיקייה `osher-calisthenics-final`.
3. העלה ל-GitHub את כל הקבצים והתיקיות שבתוכה, כולל:
   - app
   - data
   - public
   - .github
   - package.json
   - next.config.js
4. בצע Commit.
5. ב-GitHub כנס ל-Settings -> Pages.
6. Source צריך להיות GitHub Actions.
7. כנס ל-Actions וחכה שה-build יהיה ירוק.
