# Osher Calisthenics - GitHub Pages Starter

גרסת התחלה רשמית לפרויקט.

## מה יש כאן
- Next.js static export
- GitHub Actions לפרסום אוטומטי ל-GitHub Pages
- מסך בית בעברית
- מפת שרירים אינטראקטיבית
- Roadmap ל-Muscle Up

## חשוב
שם המאגר צריך להיות: `osher-calisthenics`
אם בחרת שם אחר, צריך לעדכן את `repo` בקובץ `next.config.mjs`.

## בדיקה
אחרי Upload + Commit ל-GitHub, עבור אל:
Settings > Pages > Source: GitHub Actions

ואז בדוק בלשונית Actions שההרצה הסתיימה בהצלחה.
