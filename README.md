# Osher Calisthenics - GitHub Pages Version

גרסה מעודכנת:
- ממשק שנראה כמו אפליקציה ולא דף אחד ארוך.
- ניווט תחתון בין מסכים.
- מסך בית עם גוף אינטראקטיבי ולחיצה על שרירים.
- אייקונים לכל תת קטגוריה.
- תמונת SVG לכל תרגיל.
- עמוד פירוט תרגיל.
- תוכניות, Roadmap, התקדמות ופרופיל.

## העלאה ל-GitHub
1. פתח את ה-ZIP.
2. העלה את כל התוכן לתוך הריפו שלך.
3. עשה Commit.
4. ודא ש-Settings > Pages > Source = GitHub Actions.
5. בדוק בלשונית Actions שהכל ירוק.

אם GitHub לא מעלה את התיקייה `.github`, צור קובץ ידנית בשם:
`.github/workflows/deploy.yml`
והעתק אליו את התוכן מה-ZIP.
